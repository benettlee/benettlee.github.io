Terminal界面基本操作

Terminal界面基本操作

文件操作：

  • 新建文件夹： mkdir Folder
  • 复制文件夹：cp -r Folder NewFolder
  • 移动文件夹：mv Folder NewFolder
  • 删除文佳佳：rm -r Folder
  • 进入文件夹：cd Folder
  • 进入上一层文件夹： cd ../;
  • 新建文件：touch NewFile
  • 复制文件：cp File NewFile
  • 移动文件：mv File NewFile
  • 删除文件：rm File
  • 浏览当前目录：ls
区分文件夹和文件：ls -color
详细文件信息：ls -l -t -hcolor
  • 查看命令使用说明：command –help

Terminal常用工具

  • shell语言：bash
  • 文本阅读工具：vim
  • 文本搜索工具：grep
  • 文本流编辑工具：sed/awk
